Straight to the Point Archery

Full site for a fictional archery education center in Tacoma, WA. Uses CSS Grid to feature card-like articles.

Kitties 2 Go!

A site for a fictional cat rental service. Complete with flexbox!

Semantic Elements

This project explores the purpose and use of the semantic tags that help keep HTML5 developers sane. It is the product of the first assignment of UWPCE's HTML5 100 course.

GDDNS Updater

A simple Python script to update your dynamic DNS host on our favorite not evil platform. It is written in Python3 and runs via command line.

Winner Dinner

The Winner Dinner app picks dinner for you! It shows you a semi-random recipe based on the components selected. Currently supports 9 meats, 5 carbs, and 7 cuisines!
